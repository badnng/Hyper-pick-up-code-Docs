# 识别中遇到的问题

### 问题一：识别失败或无结果

**可能原因：**
- 无取餐码内容
- 文字被遮挡或截断

**解决方案：**
确保取餐码能够截取，截图大小是裁剪过的，中上部分都能裁到

### 问题二：识别结果不准确

**可能原因：**
- 文字格式特殊
- 包含干扰信息
- OCR 识别误差

**解决方案：**
1. 尝试使用不同的识别方式
2. 手动选择更精确的文字区域
3. 避免选择无关的背景文字
4. 清理识别结果中的多余字符

## 性能相关问题

### 问题六：识别速度慢

**可能原因：**
- 手机性能较低

**解决方案：**
- 无/换个新手机


## 权限相关问题

### 问题八：权限被拒绝

**可能原因：**
- 用户手动拒绝
- 系统安全策略限制
- 权限请求被忽略

**解决方案：**
1. 手动进入设置授予必要权限
2. 检查是否有安全软件拦截
3. 重新运行引导程序
4. 参考[快速开始](./getting-started)指南

## 其他问题

### 问题九：历史记录丢失

**可能原因：**
- 清除了 App 数据
- 存储空间不足
- 数据库损坏

**解决方案：**
1. 检查是否误清除了 App 数据
2. 定期导出重要的识别记录
3. 确保设备有足够的存储空间

### 问题十：无法更新到最新版本

**可能原因：**
- 网络连接问题

**解决方案：**
- 本项目通过github release获取更新，并套了一层cloudflame加速下载，大陆地区90%都可以下载到

如仍无法下载，请自行解决
